### Руководство по прошивке ESP32

*Оригинальная документация:* [https://meshtastic.org/docs/getting-started/flashing-firmware/esp32/](https://meshtastic.org/docs/getting-started/flashing-firmware/esp32/)

**Методы прошивки для устройств ESP32**

Есть несколько доступных методов для прошивки вашего ESP32 устройства: Web Flasher либо прошивка вручную с помощью утилит esptool или flash_download_tool.

В зависимости от версии установленного MCU (ESP32, ESP32-S3, ESP32-C3, ESP32-C6 и пр.) вам может потребоваться перевести устройство в режим загрузчика перед прошивкой. В большинстве случаев необходимо подать питание на устройство при нажатой кнопке "boot". В зависимости от MCU и настроек прошивки, ручной перевод в режим загрузчика нужно будет производить только при первой, либо при каждой прошивке.
> если прошивка не начинается - попробуйте перевести в режим загрузчика вручную

**Прошивка через Web Flasher**

Самый простой способ, особенно для неподготовленных пользователей.

[Официальный флешер Meshtastic](https://flasher.meshtastic.org) или [Mrekin Meshtastic Flasher](https://mrekin.duckdns.org/flasher/)

Для прошивки требуется компьютер с установленным браузером на движке Chromium (такие как Chome, Edge, Vivaldi, Brave, Yandex и др.). 

Процесс простой:
1. необходимо выбрать из списка ваше устройство и версию прошивки
2. подключить устройство в режиме загрузчика в компьютеру
3. выбрать режим (обновление или полная прошивка)
4. нажать кнопку старта и дождаться окончания прошивки

**Обновление по воздуху (OTA)**

Обновление OTA для ESP32 устройств не работает ввиду недостаточности памяти для прошивки на устройствах.
